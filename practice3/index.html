バインディング式

  • {{ message }}
  • {{ number + 1 }}
  • {{ message.split('').reverse().join('')}}
  • {{ok ? 'YES' : 'NO'}}

filter: ローカルフィルタ

  • Vue.jsでは式の終わりに任意のフィルタを追加できる
  • {{ 式 | フィルタ名}}
  • {{price | numberFormat }}

filter: グローバルフィルタ

  • インスタンス生成前に作成
  • Vie.filter(フィルタ名、フィルタの動作)
  • {{price | numberFormat2 }}

filter: フィルタの連結

  • {{ jpyPrice | toUSD | numberFormat2}}

filter: フィルタの引数

  • {{ text | readMore(20, '...')}}

v-bindの省略記法

  • 完全な構文
  • {{ v-bind1 }}
  • 省略記法
  • {{ v-bind2 }}
